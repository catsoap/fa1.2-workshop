### Requirements

You will need python3.6 and pipenv (installed by `pip install pipenv`).
And other required libs (see [pytezos doc](https://pytezos.org/quick_start.html))

#### Local testing

- Create your pipenv environnement
  > pipenv install
- Load your pipenv
  > pipenv shell
- Run the tests
  > pytest -v .
